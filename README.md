# JS & TS Interactive Learning Flashcards

A premium, modern, and highly responsive web application designed to help absolute beginners master **JavaScript** and **TypeScript** from scratch. This project features **300 comprehensive flashcards** with elegant 3D card-flipping animations, a smooth progress indicator, and immersive deep-dive explanation modals.

## 🚀 Live Demo
Deploy this project directly to **GitHub Pages** to share your frontend portfolio link!

## ✨ Features
- **300 Beginner-Friendly Cards:** Structured progressively from basic variables to advanced TypeScript configuration.
- **Premium Glassmorphic UI:** Built with a clean, dark-accented palette, modern typography (`Noto Sans Georgian`), and fluid micro-interactions.
- **3D Flip Interaction:** Click anywhere on the card to smoothly flip between the question and the answer.
- **Deep-Dive Modals:** Click "Detailed Explanation" on any card to view extensive real-world analogies and structural deep dives.
- **Modular Codebase:** Organized architecture with dedicated data layers for optimal maintenance and scaling.
- **100% Mobile Responsive:** Perfectly optimized for desktops, tablets, and smartphones.

## 🛠️ Technologies Used
HTML5: Semantic architecture and accessibility structures.

CSS3: Custom properties (CSS variables), Flexbox layout patterns, and hardware-accelerated 3D rotation animations (perspective, backface-visibility).

Vanilla JavaScript: Clean, modern execution flow focusing on modular event handling and safe DOM binding.
