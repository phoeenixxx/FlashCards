# JS & TS Interactive Learning Hub (Flashcards & Quiz)

A premium, modern, and highly responsive web application designed to help absolute beginners master **JavaScript** and **TypeScript** from scratch. This project features **300 comprehensive, dual-language flashcards** alongside an **interactive quiz engine**, utilizing elegant 3D card-flipping animations, theme/language toggles, and dynamic question shuffling.

## ✨ Features
- **300 Dual-Language Cards:** Structured progressively from basic variables to advanced TypeScript architecture, fully accessible in both **Georgian (KA)** and **English (EN)**.
- **Dual Application Modes:**
  - **Flashcards:** Master concepts intuitively with sleek 3D card-flipping interactions and comprehensive explanations.
  - **Interactive Quiz:** Test your knowledge with dynamically generated 10-question quizzes featuring instant visual feedback, automatic distractor distribution, and post-game retry options.
- **Dynamic Shuffling Engine:** Cards are shuffled randomly every time you start a session or generate a quiz, ensuring an optimal, non-repetitive learning pattern.
- **Premium Themes (Light & Dark):** Immersive glassmorphic user interface seamlessly adaptable to light and dark system preferences.
- **Deep-Dive Modals:** Click "Detailed Explanation" on any card back to open structural breakdowns, contextual examples, and real-world analogies.
- **100% Mobile Responsive:** Optimized layout flow across all devices—desktops, tablets, and smartphones.

## 🛠️ Technologies Used
HTML5: Semantic document layout, data attribute binding, and accessibility structures.

CSS3: Custom variables (CSS properties) for theme configuration, custom scrollbars, hardware-accelerated 3D transforms (perspective, backface-visibility), and responsive Flexbox grids.

Vanilla JavaScript (ES6+): Scalable, comment-free clean code architecture managing localized state, dynamic template string injections, and array randomization algorithms (Fisher-Yates shuffle).
